Go Study App

This application is an auto-saving locally hosted Go Study App.

Use the tooltip toggle to turn tooltips on and off.

The Proverbs widget diplays a random proverb you have saved at initiation, and clicking next will cycle through proverbs.

If you click search you can search for proverbs containing a keyword.One will be diplayed, and clicking next will show additional proverbs that fit the search until there are no more that fit the search.

The Notes widget displays your current notes, with a number of buttons for editting and updating them.

You can make a note into a proverb, but it will still be saved as a note until you delete it from your notes.

This application as a whole auto-saves changes you make.

The news widget displays current Go news headlines from the American Go Association Website.

In the news widget you can click next or previous (previous is a new feature, try it!)

Make sure to run the GoStudyNodeV2.js with nodejs in order to use this app!
